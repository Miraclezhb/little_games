# 项目介绍
`little_games` 顾名思义，就是一些小游戏合集，希望通过重写一些经典的小游戏，提高自己对代码的理解把握。
如果这个项目能帮助到其他人，那就再好不过了。
目前打算使用`java`开发，后续考虑其他语言的版本

## 环境需求
> JRE>=17
